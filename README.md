# tennis-match-tool
Interactive Python tool simulating an amateur tennis match for two players, using simplified mixed doubles rules

Description: Interactive Python tool simulating an amateur tennis match for two players, using simplified mixed doubles rules.
Tracks points, games, sets, and tie-breaks in real-time

Rules/Features: Best-of-3 sets format with a 10-point super tie-break if sets are tied
Standard tennis scoring: 0, 15, 30, 40 with No-Add simplification
Live scoreboard showing points, games, and sets in a clear table-like format
Interactive input for point winners and automatic server rotation
Simple, beginner-friendly code structure using lists, loops, and conditional logic

Programing language: Python
User language: English

How to run: Copy the code and paste it in IDE
Run the script and register player names
The program flips a coin to choose the first server
Input the winner of each point when prompted
Watch the live scoreboard update after each point, game, and set
The match ends with the announcement of the winner

Purpose: A learning project demonstrating Python fundamentals while simulating a real-world scenario in an engaging, interactive way.
This tool has been used by my local club, for their Amateur Tennis ladder tournament! (Fyi I participate!!!)
